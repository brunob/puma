Information

Where does the data come from?

Publication Data

Data and metadata for the publications are downloaded using the PubMed and DOI APIs.

Citations Data

Citation data is retrieved from the Scopus API provided by Elsevier and from the Europe PMC API.

Geodata

The coordinate location and town of institutions is retrieved from the free project Wikidata.

Why don't some statistics use data from all publications?

Data can be missing because some publications are very old. There are many different ways to track publications and prior to the introduction of DOIs in 2000 there was no standard method. This means some metadata on old publications could have been lost or not recorded.

The data used for the statistics are gathered from databases which only collect data from particular journals. This problem is most obvious for citation data from Scopus and Europe PMC. Although they have a particular publication on record, there may be citations for this publication from a journal that they do not index and therefore these will not be in the citation count. This is why citiation counts from different sources are not always the same.
